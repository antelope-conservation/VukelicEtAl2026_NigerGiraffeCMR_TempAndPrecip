# VukelicEtAl2026_NigerGiraffeCMR_TempAndPrecip
This repository contains supporting data and R code for the capture–mark–recapture (CMR) analysis used to assess the effects of temperature and precipitation on apparent survival of the West African giraffe in the Giraffe Zone from 2005 to 2018, implemented using multistate mark–recapture (MSMR) models.
